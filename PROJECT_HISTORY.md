# 프로젝트 개발 내역 (Project History)

## 요약

**nuucard** MVP 개발이 완료되었으며, 최적화 및 검증까지 마쳤습니다. 코드는
깔끔하게 정리되었고, 모든 핵심 기능이 정상 작동합니다.

## 주요 구현 기능

### 1. 랜딩 페이지 UX 개편

- **히어로 비디오 (Hero Video)**:
  - **동적 멀티 비디오 지원**: `public/images` 폴더에 파일을 넣기만 하면
    자동으로 인식합니다. (코드 수정 불필요)
  - **서버 사이드 로딩 (SSR)**: 페이지 로딩 시 서버에서 즉시 비디오 목록을
    가져와 **깜빡임 없이** 재생됩니다.
  - **랜덤 재생 & 이어보기**: 접속 시 랜덤으로 영상을 재생하고, 끝나면 다음
    영상으로 자연스럽게(썸네일 없이) 이어집니다.
  - **반응형 디자인**: 모바일은 꽉 찬 화면(`object-cover`), 데스크탑은 전체
    화면(`object-contain`)으로 최적화했습니다.
  - **워터마크 숨김**: CSS Zoom(`scale-110`)과 하단 그라데이션 오버레이를 통해
    AI 워터마크를 깔끔하게 가렸습니다.
- **리드 마그넷 (Lead Magnet)**: 배경화면 다운로드 및 인스타그램 링크 섹션을
  추가하여 이탈을 방지합니다.
- **작가 모집 섹션**: 계층 구조를 낮추어 부담 없이 접근하도록 디자인을
  개선했습니다.
- **FOMO 카피라이팅**: "지금 전하지 않으면..." 등의 문구로 행동을 유도합니다.

### 2. 핵심 기능 (Core Functionality)

- **편지 쓰기**: Turnstile 보안이 적용된 암호화 편지 작성 기능.
- **편지 보기**: URL 해시를 통한 복호화 및 "실물 카드로 소장하기" 옵션 제공.
- **데이터 분석**: GA4, Microsoft Clarity, Cloudflare Web Analytics 연동 완료.

## 기술적 개선 사항 (Technical Improvements)

- **아키텍처 개선**: `page.tsx`를 **Server Component**로 전환하고, UI 로직을
  `LandingPageClient.tsx`로 분리하여 성능을 극대화했습니다.
- **컴포넌트 분리**: `HeroVideo` 컴포넌트를 만들어 비디오 재생 로직을 깔끔하게
  관리합니다.
- **유틸리티 추가**: `src/lib/videos.ts`를 통해 파일 시스템 스캔 로직을
  모듈화했습니다.
- **Git 정리**: 중첩된 리포지토리 문제를 해결하여 버전 관리를 정상화했습니다.
